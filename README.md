# Rés0sem

This repository accompagny the paper _Agronomic Performance and Microbial Diversity of Wheat Following Organic and Synthetic Seed Treatments: A Three-Year On-Farm Swiss Field Study_, which summaries some of the results of the the [Rés0sem project](https://resosem.ch/). By studying the effect of seed treatment (both chemical and organic alternatives) on both the microbiome and crop yield, Rés0sem aims at providing a scientific basis to reduce seed pesticide treatments.

The project is funded by the [Federal Office for Agriculture](https://www.blw.admin.ch/en) (Switzerland) and supported by various partners

- [Canton de Vaud](https://www.vd.ch/)
- [Canton du Valais](https://www.vs.ch/)
- [Proconseil](https://www.prometerre.ch/unites/proconseil)
- [ASS - Société coopérative des sélectionneurs](http://ass-agri.ch/)

In this paper, researchers from two institutions have collaborated:

- [FiBL](https://www.fibl.org/en/)
- [Agroscope](https://www.agroscope.admin.ch/agroscope/en/home.html)

with the support of many producers, who have actively participated in the data collection process.

## Agronomic dataset

The data set is accessible on [Zenodo](https://zenodo.org/records/19335779/files/Res0sem_agronomy_data.csv?download=1). It contains four variables of interest:


- Yield
- Proteine
- Thousand grain weight (gr)
- Average number of plants/meter 

In addition to those variables, we have information about the treatment, the region, the production mode (organic, without fungicide or conventional) and the year, as well as an identifier for the farm. The precise location of the farms are not made available for privacy reasons. See the [Zenodo](https://zenodo.org/records/19335779) page for a detailed description of the data set.
